# Prism Creative

![Preview](https://cdn.poehali.dev/templates/prism-creative/preview-ru.jpg)

Лендинг для креативного агентства с портфолио работ. Включает секции: герой-блок с анимациями, витрину проектов, логотипы клиентов, статистику, услуги, кейсы, отзывы, описание подхода, тарифы, FAQ и CTA. Построен на React + Tailwind + Framer Motion.
